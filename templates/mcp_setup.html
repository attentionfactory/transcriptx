Setup guide

Use TranscriptX inside your AI client

TranscriptX is an MCP server. Generate a personal token, paste the URL into your AI client's MCP config, and say "transcribe this video" inside Claude Desktop, Cursor, ChatGPT, or any other MCP-compatible client.

Generate your token →

Replace YOUR_TOKEN below with the token from your account page. The full URL form is https://mcp.transcriptx.xyz/?token=YOUR_TOKEN.

Claude Desktop

Open ~/Library/Application Support/Claude/claude_desktop_config.json on Mac, or %APPDATA%\Claude\claude_desktop_config.json on Windows. Add to the mcpServers block: 

{
  "mcpServers": {
    "transcriptx": {
      "url": "https://mcp.transcriptx.xyz/?token=YOUR_TOKEN"
    }
  }
}

Restart Claude Desktop. Try: "Transcribe this and pull out the three best quotes: <YouTube URL>"

Cursor

Settings → MCP → Add new server.

  • Name: TranscriptX
  • Type: streamableHttp (or http on older versions)
  • URL: https://mcp.transcriptx.xyz/?token=YOUR_TOKEN

ChatGPT (with MCP support)

In ChatGPT's connector settings, add a new MCP server. URL: https://mcp.transcriptx.xyz/?token=YOUR_TOKEN. Auth: None (token is in the URL).

Continue.dev

In your Continue config (typically ~/.continue/config.json): 

{
  "mcpServers": {
    "transcriptx": {
      "url": "https://mcp.transcriptx.xyz/?token=YOUR_TOKEN"
    }
  }
}

Cline

Cline → MCP Servers settings:

{
  "transcriptx": {
    "url": "https://mcp.transcriptx.xyz/?token=YOUR_TOKEN"
  }
}

Authorization header (alternative)

If your client doesn't accept the token in the URL, use:

  • URL: https://mcp.transcriptx.xyz/
  • Header: Authorization: Bearer YOUR_TOKEN

Both are equivalent — same token, same DB lookup.

What you can do once it's connected

Three tools become available to your AI:

  • transcribe_url — transcribe any public video or audio URL
  • list_recent_transcripts — your recent transcript history
  • get_transcript_by_id — fetch metadata for a past transcript

Try things like:

  • "Transcribe this YouTube video and give me three pull quotes."
  • "Summarize this Loom in five bullets: <URL>"
  • "Translate this Spanish video transcript to English."

Common things that break

  • "Connection refused" or 401. Token is wrong, expired, or revoked. Generate a new one.
  • Tools don't appear, no error. Restart your AI client fully. Some clients cache the tool list per-session.
  • "Credit limit reached." Same monthly cap as the web product. Upgrade to Pro for unlimited, or wait for the monthly reset.
  • Transcript came back wrong language. Use the language retry on the web for now (how it works). The MCP-native retry tool ships in v0.5.
